# 🧹 Limpieza AsoUsados

Script de Visual Basic para la automatización y estandarización de la limpieza de datos en archivos [tipo de archivo: Excel]


## ✨ Funcionalidades

Este repositorio contiene las herramientas esenciales para el proceso de limpieza y preparación de datos:

* **Script de Limpieza de Datos:** Un archivo escrito en **Visual Basic** diseñado para realizar tareas de limpieza de datos como:
    
* 	Seleccionar - El usuario elige el rango a limpiar
*	Buscar - Revisa cada celda buscando texto blanco
*	Limpiar - Borra solo el contenido de las celdas con texto blanco

* **Guía de Uso (Documento Word):** Un documento editable que actúa como manual detallado para la configuración y ejecución del script, así como para la comprensión de los criterios de limpieza aplicados.

## 🛠️ Tecnologías Utilizadas

* **Visual Basic (VBA o VB.NET):** Lenguaje de programación utilizado para el desarrollo del script de limpieza.
* **Microsoft Word:** Usado para la documentación de la guía de usuario.
